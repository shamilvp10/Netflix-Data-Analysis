# 📊 Netflix Data Analysis

## 📂 Dataset

* Dataset used: Netflix Titles Dataset
* Source: Kaggle
* Link: [https://www.kaggle.com/datasets/shivamb/netflix-shows?resource=download](https://www.kaggle.com/datasets/shivamb/netflix-shows?resource=download)
* Contains Movies and TV Shows data (~8,800 records)

---

## 🎯 Project Aim

* To analyze Netflix content
* To understand trends in movies and TV shows
* To learn data cleaning and visualization

---

## 🧹 Data Cleaning

* Handled missing values
* Cleaned columns (director, cast, country)
* Fixed duration column
* Converted data into proper format

---

## 📊 Analysis Performed

* Content distribution (Movies vs TV Shows)
* Release year trend
* Genre analysis
* Country analysis
* Ratings analysis

---

## 📈 Visualizations

* Histogram
* Box Plot
* Heatmap
* Pair Plot

---

## 📌 Key Insights

* Movies are more than TV Shows
* Most content released after 2010
* Popular genres: Drama and Comedy
* Top countries: USA, India, UK
* Most content is for teens and adults

---

## ✅ Conclusion

* Dataset was cleaned and analyzed successfully
* Netflix focuses on modern content
* Learned data cleaning and visualization techniques
